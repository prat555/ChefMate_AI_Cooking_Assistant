## Overview

ChefMate is an intelligent cooking assistant that leverages artificial intelligence to make cooking more accessible and enjoyable. The application combines a modern React frontend with a Python FastAPI backend, utilizing Pydantic AI for intelligent recipe generation, ingredient substitution recommendations, and personalized cooking guidance.

The platform addresses common cooking challenges by providing instant access to recipe suggestions based on available ingredients, offering smart ingredient alternatives when items are missing, and delivering expert cooking advice through an interactive chat interface. Built with a focus on usability and performance, ChefMate uses OpenRouter's free AI models to deliver fast, accurate responses without requiring payment.

## Core Features

**AI Chat Interface**: Interactive conversational assistant that answers cooking questions, provides technique guidance, explains culinary concepts, and offers personalized recommendations. Users can ask about cooking methods, troubleshooting, timing, and best practices.

**Ingredient-Based Recipe Search**: Intelligent recipe discovery system that analyzes available ingredients and dietary restrictions to suggest complete recipes with detailed instructions, cooking times, and difficulty levels. The system considers ingredient combinations and dietary preferences to generate practical meal options.

**Smart Substitution Engine**: Context-aware ingredient replacement system that provides alternatives for missing or restricted ingredients. The engine considers the recipe context, provides conversion ratios, explains flavor impacts, and suggests multiple options ranked by suitability.
